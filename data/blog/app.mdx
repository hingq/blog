# Mac 常用效率应用推荐

日常开发和写作里，真正会长期留下来的工具，通常不是功能最多的那一类，而是启动快、负担小、能稳定解决问题的那一类。

这篇文章整理几款我在 Mac 上比较常用的应用，覆盖本地容器、接口调试、Markdown 编辑、请求拦截和远程连接几个高频场景。如果你也希望把工作流从“功能堆叠”切换到“够用且顺手”，这些工具值得试一遍。

## 为什么是这几款应用

我对工具的筛选标准比较直接：

1. 能快速进入工作状态，不需要复杂配置。
2. 日常使用成本低，不会明显拖慢系统。
3. 能替代更重、更复杂的同类产品。
4. 在一个明确场景里足够好用，而不是追求面面俱到。

下面这几款工具，基本都符合这几个标准。

## 1. OrbStack

OrbStack 是一款面向 macOS 的轻量级容器和 Linux 虚拟机工具，很多场景下可以作为 Docker Desktop 的替代方案。

它最明显的特点是占用更低、启动更快，日常跑容器时对系统资源的压力相对更小。如果你经常在本地启动数据库、缓存服务、消息队列或者完整的开发环境，OrbStack 的体验通常会比传统方案更轻一些。

它适合这类场景：

- 本地需要频繁运行 Docker 容器。
- 希望减少 Docker Desktop 带来的资源占用。
- 更在意启动速度和系统整体流畅度。

如果你的核心诉求是“在 Mac 上稳定、低开销地运行容器”，OrbStack 很值得作为默认选择。

官网：[https://orbstack.dev/](https://orbstack.dev/)

## 2. Yaak

Yaak 是一个轻量的 API Client，适合用来做日常接口调试和请求测试。

相比功能非常重的一体化平台，Yaak 的优势在于界面更干净、使用路径更短，打开之后可以很快开始调试接口。对于只想专注在请求发送、响应查看、环境变量切换这些基础能力上的人来说，它会比一些“大而全”的工具更省心。

它适合这类场景：

- 本地调试 REST API、GraphQL 或常规 HTTP 请求。
- 不需要把接口文档、团队协作、测试平台全部绑在一个工具里。
- 更喜欢轻量、专注的桌面客户端。

如果你已经觉得某些 API 调试工具功能太多、界面太重，Yaak 会是一个更轻的选择。

官网：[https://yaak.app/](https://yaak.app/)

## 3. Typora

Typora 是一款非常经典的 Markdown 编辑器，适合写博客、记笔记和整理技术文档。

它的优点在于编辑体验非常流畅，界面干净，写作时不容易被多余的按钮和面板打断。对于习惯使用 Markdown 的人来说，Typora 基本属于“打开就能写”的工具，不需要额外适应太久。

它适合这类场景：

- 写博客正文、项目文档或个人笔记。
- 希望在源码模式和预览效果之间保持更自然的编辑体验。
- 想要一个轻量、专注文本内容的写作环境。

如果你的工作里有不少文字输出，Typora 依然是一个很稳的选择。

官网：[https://typora.io/](https://typora.io/)

## 4. Requestly

Requestly 是一个非常实用的请求拦截和重写工具，适合前端调试、联调排查和本地模拟。

它可以帮助你对请求做重定向、Header 修改、参数替换、延迟注入、Mock 响应等处理。在接口还没准备好、后端环境不稳定，或者你需要快速验证某个异常场景时，这类能力会非常省时间。

它适合这类场景：

- 前端联调时临时改写接口请求。
- 模拟接口异常、超时或特定返回内容。
- 排查线上问题时复现某些网络层行为。

如果你经常处理接口联调和浏览器请求调试，Requestly 会比手写临时脚本更直接。

官网：[https://requestly.com/](https://requestly.com/)

## 5. Termius

Termius 是一款现代化的远程连接工具，主要用于 SSH 主机管理。

相比系统自带终端直接连服务器，Termius 在主机管理、连接配置、密钥管理和多设备同步这些方面会更方便一些。对于需要频繁连接多台服务器、测试机或云主机的人来说，它能明显降低记忆成本和切换成本。

它适合这类场景：

- 需要长期维护多台远程服务器。
- 希望统一管理 SSH 配置和密钥。
- 会在 Mac、iPad、手机等多个设备之间切换远程连接。

如果你平时远程操作主机的频率比较高，Termius 会比手动维护一堆终端配置更省事。

官网：[https://termius.com/](https://termius.com/)

## 总结

如果按使用场景来分，这几款工具可以分别覆盖一条很清晰的工作流：

- `OrbStack` 负责本地容器环境。
- `Yaak` 负责接口调试。
- `Typora` 负责 Markdown 写作。
- `Requestly` 负责请求拦截和联调辅助。
- `Termius` 负责远程连接和主机管理。

它们不一定都是同类产品里功能最多的，但都属于“打开就能用、长期用起来不累”的工具。如果你正在逐步整理自己的 Mac 工作环境，这几款应用都值得放进候选名单里。
